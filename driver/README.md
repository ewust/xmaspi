xmaspi network service
======================

Drive your RGB Christmas lights from UDP or TCP

Requires `/dev/xmas` and libevent2.

It is super cool!

Build
-----

To compile, you'll need:
    
    `apt install libevent-dev`

Then run:

    `make udp-driver`

or

    `make tcp-driver`

Run
---

Use the `-v` flag to output performance counters.
